## Concept

An interactive XR environment where user input (voice, text, or presence) dynamically shapes visual and sensory feedback in real time.

## Core Idea

Instead of static XR experiences, this project explores adaptive environments that respond to human state.

## States

- Idle → baseline awareness
- Calm → breathing / grounding
- Fire → intensity / activation

## Tech Direction

- Unity (XR-ready)
- Real-time state system
- Future layers:
  - Voice input
  - Prompt-based interaction
  - Adaptive feedback systems

## Goal

Build a working prototype during XRCC 2026 that demonstrates how XR + AI can create responsive, living environments.
